Git 的简单入门

发表于2021-01-04|更新于2021-04-22|技术
|字数总计:1.3k|阅读时长:4分钟

在使用 Hexo 搭建本博客的过程中已经使用过 Git，相信大家对它已不陌生。Git 是最常用的分布式版本控制工具，实际开发一般都会用到，所以很有必要学习。

前言

Git 分为本地仓库(repository)和远程仓库(remote)，我们可以在本地和远程间提交和下载；本地仓库分为我们设置为仓库的文件夹——工作目录（Working Tree），因为并不一定每个文件都需要进行版本控制所以设置的缓冲区——索引，以及我们本地的版本库。

每个版本只会存储我们进行的修改，而不是完整的文件。

本文参考这个很棒的入门网站：猴子都能懂的 GIT 入门
